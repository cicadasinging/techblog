## Steps

### 云端操作（Github）：

1. 创建远程仓库 `demo-vite-tailwindcss--weather`

### 本地操作（WebStorm）：

1. 打开【Get from Version Control】弹窗：
    - 选择【GitHub】
    - 登录【Log ln to GitHub】
    - 选择远程仓库 `demo-vite-tailwindcss--weather`
    - 点击【Clone】
2. 打开【Terminal】边窗：
    - [create-vue](https://github.com/vuejs/create-vue)

```bash
npm create vue@3
```
