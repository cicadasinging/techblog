# demo-vite-with-blog

https://github.com/cicadasinging/demo-ts-with-blog

## Steps

### 云端操作（Github）：

1. 创建远程仓库 `demo-ts-with-blog`

### 本地操作（WebStorm）：

1. 打开【Get from Version Control】弹窗：
    - 选择【GitHub】
    - 登录【Log ln to GitHub】
    - 选择远程仓库 `demo-ts-with-blog`
    - 点击【Clone】

2. 打开【Terminal】边窗：

```bash
yarn init
yarn create vite
```
